# QuickSort en Java

## Descripción
Implementación de QuickSort en Java utilizando particionado Lomuto y pivote aleatorio.
Se probaron escenarios: ordered, reversed, random y nearlySorted con varios tamaños (n).
Se midieron comparaciones, swaps y tiempo (ns). Los resultados se graficaron en PDF.
